Estructura de control = conjunto mínimo de instrucciones.
|
|
----> - read, write
      - if, case
      - while, repeat until
      - for

Programas en Pascal: {
    Program nombre;
    Const
        ... (Constantes del programa)
    módulos
        ... (Se verá mas adelante)
    var
        ... (Variables del programa)
    Begin (Inicialización)
        ...
    end. (Finalización)
};

Ejemplo de programa estructurado -> {
    Program ejemplo;
    Const (durante toda la ejecución, el valor se mantiene)
        N = 25;
        pi = 3,14;
    var
        edad: integer;
        peso: real;
        letra: char;
        resultado: boolean;
    Begin
        edad := 5;
        peso := 63,5;
        edad := edad + N; (siendo 30 por Const)
        leta := 'A';
        resultado := letra = 'A'; (como A y a no son iguales, devuelve dato false (boolean definido en var))
    end.
};

# PRE y POST CONDICIONES

    - Pre-condición: La información que se conoce como verdadera ANTES de iniciar el programa (o módulo)
    - Post-condición: La información que debería ser verdadera al concluír el programa (o módulo) si se cumplen
                      adecuadamente los pasos especificados.

    READ ---
            |__ Operación que contienen la mayoría de los lenguajes de programación. Se utiliza para tomar datos
                desde un dispositivo de entrada (por defecto el teclado) y asignarlos a las variables correspondientes.
    
    Asignación de valores {
        Hasta ahora sabemos que para asignar un valor se utiliza := pero podemos utilizar la estructura
        de control "read(x)"

        Por ejemplo: {
            var
                suma, valor, edad: integer;
            Begin
                read(edad);     (el programa no avanzará hasta que se incluya un valor entero en la consola)
                valor := 9;
                suma := edad + valor;
            end.
        };
    };

    WRITE ---
            |__ Operación que contienen la mayoría de los lenguajes de programación. Se utiliza para mostrar el
                contenido de una variable.

    Tiene 4 posibilidades = {
        - Para enseñar solo texto: write("texto")
        - Para enseñar solo una variable: write(variable)
        - Para enseñar un conjunto de texto y variable: write("texto", variable)
        - Para enseñar PEQUEÑAS operaciones: write(num + 4)
    } ¿Como utilizarlo? {
        var
            cant: integer;
        Begin
            read(cant);
            cant := cant + 1;
            write(cant);
        end.
    }; 

    SECUENCIA ---
                 |__ Representada por una sucesión de operaciones (por ej. asignaciones) en la que el orden de
                     ejecución coincide con el orden físico de aparición de las instrucciones.

    Program sec;
    var
        num: integer;
    Begin
        read(num);
        write(num);
    end.

    DECISIÓN ----
       |          |__ Algorítmo representativo de un problema real cuando es necesario tomar decisiones en función de
       |              los datos del problema.
    IF/ELSE           La estructura básica se representa simbolicamente de esta manera:

                                            Código anterior
                                                   |
                                                   |
                                Verdadera <-- Condición --> Falsa
                                   |                          |
                                Acciones                   Acciones

    Sintaxis:
    (TIP sintaxis: la acción anterior a "else" NO lleva ";")

    Cuando  el "else" tiene una acción si la cond es falsa {
        if (condición) then
            acción1
        else
            acción2;
    } else {
        Si en el "if" quiero más de una acción {
            if (condición) then
                begin
                    acción1;
                    acción2;
                end
            else
                acción3
        } else {
            Si en ambas es más de una acción {
                if (condición) then
                    begin
                        acción1;
                        acción2;
                    end
                else
                    begin
                        acción1;
                        acción2;
                    end;
            },
        },
    };

    SELECCIÓN -----
       |          |__ Permite realizar distintas acciones dependiendo del valor de una variable de tipo ordinal. La
    CASE OF           estructura básica se representa simbolicamente así:

                                            Código anterior
                                                   |
                                                   |
                                Condición3 <-- Condición1 --> Condición2
                                   |                |             |
                                Acciones        Acciones       Acciones

    Sintaxis:
    case (variable) of 
        condición1:
            acción1;
        condición2:
            acción2;
    end;

    ITERACIÓN ---
                 |__ Puede ocurrir que se desee ejecutar un bloque de instrucciones desconociendo el numero exacto de
                      veces que se ejecuta. Para esto, existe la iteración en la mayoría de los lenguajes de programación. Se clasifican en PRE y POST condicionales.

    Iteración PRE condicional = Evalúan la condición y si es verdadera se ejecuta el bloque de acciones. Se puede
        |                       ejecutar 0 (falsa, no se ejecuta), 1 (verdadera una vez) y/o mas veces.
     WHILE DO                   Se representa simbolicamente de esta manera:

                                            Código anterior
                                                   |
                                                   |
                                Verdadera <-- Condición --> Falsa
                                   |               |         |
                                Acciones __________|       Acciones

    Sintaxis: {
    while (condición) do
        acción;          
    } mas de 1 {
        while (condición) do
            begin
                acción1;
                acción2;
            end;
    }

        - Si nunca leo otro producto, el while toma el primero y nunca finaliza.

    }

    Iteración POST condicional = Ejecutan las acciones, luego evalúan la condición y ejecutan las acciones mientras
        |                        la condición es falsa.
    REPEAT UNTIL                 Se representa simbolicamente de esta manera:

                                            Código anterior
                                                   |
                                                   |
                                               Acciones_____________
                                                   |                |  
                                 Verdadera <-- Condición --> Falsa  |
                                    |              |                |
                                 Acciones          |________________|

    Sintaxis: {
        repeat
            acción;
        until (condición);
        (no se utiliza begin/end si hay mas de 1)
    }

    REPETICIÓN ----
        |          |__ Extensión natural de la secuencia. Consiste en repetir N veces un bloque de acciones. Este N de
       FOR             veces ya debe ser sabido de antemano.
                       Se representa simbolicamente de la siguiente manera:

                                            Código anterior
                                                   |
                                                   |
                                  Sí <-- ¿Se repitió N veces? --> No
                                   |               |              |  
                              Otras acciones       |__________ Acciones

    Sintaxis: {
        for indice := valor_inicial to valor_final do
            acción1;
    } mas de 1 acción {
        for indice := valor_inicial to valor_final do
            begin
                acción1;
                acción2;
            end;
    }
