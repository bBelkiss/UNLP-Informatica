# Tipos de datos

Datos = Clase de objetos ligados a un conjunto de operaciones para crearlos y manipularlos.
|
|
|__ Clasificación: - Simples = Aquellos con valor único, definidos por programador o lenguaje.
                   - Compuestos = Aquellos con colección de valores.

Simples: {
    Dato numérico = Representa el conjunto de números que se puedan necesitar (enteros o reales)
        Datos enteros = Simples, ordinales / Toman valores sin decimal / Tienen máximo y mínimo
            Operaciónes válidas: - Matemáticas (+, -, *, /)
                                 - Lógicas (<, >, <=, >=)
                                 - Enteros (DIV, MOD) {
                                    MOD: Resto entero de la división.
                                    DIV: Cociente entero de la división.
                                    Ej: a := 22
                                        b := 6
                                        c := a DIV b; 3
                                        d := d MOD b; 4
                                 }
        Datos lógicos = Simples, permiten representar datos que toman valor de verdad o falsedad, forma 'true' y 'false'.
            Operaciones válidas: - Lógicas (and (conjunción), or (disyunción), not (negación)) {
                                     And    Or    Not
                                    V V V | V V V | V F        
                                    F F F | F F F | F V
                                    V F F | V F V 
                                    F V F | F V V
                                }
        Dato caracter (char) = Simple, ordinal, representa un conjunto finito y ordenado de caracteres que la computadora reconoce, valores { especiales, letras, números } un único caracter.
            Operaciones válidas: - Lógicas (<, >, <=, >=, <>, =)
        Dato String = Colección de caracteres (max = 256)
            Operaciones válidas: - Lógicas (<, >, <=, >=, <>, =, y mas pero no se dan en esta materia)  
    }

Variable = Puede cambiar su valor / Constante = Solo puede tomar un valor

# Máximos y Mínimos

Para declarar una variable máxima se debe inicializar con un valor mínimo, de modo contrario, al declarar una variable mínima
se debe tomar un valor máximo.

Máximo como variable {
    Ejercitación ejemplo = Realizar un programa que lea un número de un alumno y promedio hasta llegar a un promedio igual a 0.
                           Informar alumno con promedio mas alto. {
                                (Resolución: Primer año\1erCuatrimestre\CADP\Teorías\2.3Max.pas)
                           }
        Tomando este ejemplo, utilizamos como valor inicial '-1' ya que el promedio mínimo que puede tener un alumno es 0.
    }
Mínimo como variable {
    Ejercitación ejemplo = Al igual que el ejercicio anterior, solo que con promedio mínimo. {
                                (Resolución: Primer año\1er Cuatrimestre\CADP\Teorías\2.3Min.pas)
                            }
        Tomando este ejemplo, utilizamos como valor inicial '11' ya que el promedio máximo que puede tener un alumno es 10.                    
    }

# Tipos de datos definidos por usuario

Subrango = Datos simples, consiste en una sucesión de valores.
        
        ¿Como declararlo en Pascal? {
